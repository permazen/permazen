    public class SchemaGeneratorTask
extends org.apache.tools.ant.Task
    Ant task for schema XML generation and/or verification.

    This task scans the configured classpath for classes with @JSimpleClass and @JFieldType annotations and either writes the generated schema to an XML file, or verifies the schema from an existing XML file.

    The following tasks are supported:

    Attribute Required? Description
    mode No

    Set to generate to generate a new XML file, or verify to verify an existing XML file.

    Default is verify.

    file Yes

    The XML file to generate or verify.

    matchNames No

    Whether to verify not only schema compatibility but also that the two schemas are identical, i.e., the same names are used for object types, fields, and composite indexes.

    Two schemas that are equivalent except for names are considered compatible, because the core API uses storage ID's, not names. However, if names change then some JSimpleDB layer operations, such as index queries and reference path inversion, may need to be updated.

    Default is true. Ignored unless mode is verify.

    failOnError No

    Whether to fail if verification fails.

    Default is true. Ignored unless mode is verify.

    verifiedProperty No

    The name of an ant property to set to true or false depending on whether verification succeeded or failed. Useful when failOnError is set to false and you want to handle the failure elsewhere in the build file.

    Default is to not set any property.

    classpath or classpathref Yes

    Specifies the search path for classes with @JSimpleClass annotations.

    packages Yes

    Specifies one or more Java package names (separated by commas and/or whitespace) under which to look for classes with @JSimpleClass and @JFieldType annotations.

    storageIdGeneratorClass No

    Specifies the name of an optional custom StorageIdGenerator class.

    By default, a DefaultStorageIdGenerator is used.

    Example: 

      <project xmlns:jsimpledb="urn:org.dellroad.jsimpledb" ... >
      ...
      <taskdef uri="urn:org.dellroad.jsimpledb" name="schema"
        classname="org.jsimpledb.ant.SchemaGeneratorTask" classpathref="jsimpledb.classpath"/>
      <jsimpledb:schema mode="verify" classpathref="myclasses.classpath"
        file="expected-schema.xml" packages="com.example.model"/>
    See Also:
    JSimpleDB, SchemaModel
