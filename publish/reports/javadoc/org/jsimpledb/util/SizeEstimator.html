org.jsimpledb.util

Class SizeEstimator

  • java.lang.Object
    • org.jsimpledb.util.SizeEstimator

  • public class SizeEstimator
extends Object
    Used to estimate Java object memory usage in bytes.

    For classes that contain references to other objects, what exactly is included in an object's "size" is defined on a case-by-case basis.
