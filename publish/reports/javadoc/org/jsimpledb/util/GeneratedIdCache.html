      • getGeneratedId

        public ObjId getGeneratedId(Transaction tx,
                   int storageId,
                   String string)
        Generate an object ID for the given object type storage ID and string.

        If an ObjId has already been created for the specified storageId and string by this instance, it will be returned. Otherwise, a random ObjId for which no object exists in the specified Transaction is generated and returned.

        Parameters:
        tx - transaction from which to allocate new object IDs
        storageId - object type storage ID
        string - unique string
        Returns:
        unique, unallocated object ID corresponding to storageId and string
        Throws:
        IllegalArgumentException - if storageId is invalid
        IllegalArgumentException - if tx or string is null
