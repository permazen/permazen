      • AUTOMATIC

        public static final ValidationMode AUTOMATIC
        Objects are enqueued for validation automatically.

        In this mode, objects are enqueued for validation whenever JObject.revalidate() is invoked, or automatically when:

        • An instance is created, and the Java model type (or any super-type) has a JSR 303 or @Validate annotation on itself or a public method
        • An instance is upgraded, and the Java model type (or any super-type) has a JSR 303 or @Validate annotation on itself or a public method
        • A database field is modified, and the corresponding Java model `getter' method has any JSR 303 annotations

        Note that the presence of a @Validate annotation on some method does not in itself result in automatic validation when any field changes. To achieve that, add an @OnChange-annotated method that invokes this.revalidate().
