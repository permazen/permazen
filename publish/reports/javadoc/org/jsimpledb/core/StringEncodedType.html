    public class StringEncodedType<T>
extends NullSafeType<T>
    A FieldType implementation for any Java type that can be encoded uniquely as a String. A Converter is used to convert between native and String forms.

    This class provides a convenient way to implement custom FieldTypes. Null values are supported and null is the default value. This type will sort instances according to the lexicographical sort order of their String encodings; null will sort last.
