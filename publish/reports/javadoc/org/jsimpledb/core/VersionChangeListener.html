      • onVersionChange

        void onVersionChange(Transaction tx,
                   ObjId id,
                   int oldVersion,
                   int newVersion,
                   Map<Integer,Object> oldFieldValues)
        Receive notification of an object schema version change.

        Notifications are delivered in the same thread that first reads the object, before the operation that triggered the schema version change returns.

        Parameters:
        tx - associated transaction
        id - the ID of the updated object
        oldVersion - previous object schema version
        newVersion - new object schema version
        oldFieldValues - the values of all fields in the old schema version keyed by storage ID
