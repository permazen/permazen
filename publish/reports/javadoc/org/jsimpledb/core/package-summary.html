Package org.jsimpledb.core Description

Core classes for the JSimpleDB Java persistence layer.

The classes in this package implement the core JSimpleDB engine. This layer can function entirely independently from the annotation-based JSimpleDB Java object model layer; the latter is implemented on top of these classes.
