org.jsimpledb.core

Class Schemas

  • java.lang.Object
    • org.jsimpledb.core.Schemas

  • public class Schemas
extends Object
    Contains the set of all Schemas of objects visible in a Transaction. All database objects have an implicit schema version number, which corresponds to one of the Schemas contained here and describes the object's fields.

    Associated with a Transaction is a specific schema version that the transaction uses to access objects.
