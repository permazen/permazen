org.jsimpledb.core

Class SnapshotTransaction

  • java.lang.Object
    • org.jsimpledb.core.Transaction
      • org.jsimpledb.core.SnapshotTransaction

  • public class SnapshotTransaction
extends Transaction
    An initially empty, in-memory Transaction that persists indefinitely.

    SnapshotTransactions hold a "snapshot" some portion of the state of a Transaction for later use. Each SnapshotTransactions contains its own set of "snapshot" objects that are copies of corresponding "real" database objects (as of the time they were copied).

    SnapshotTransactions can never be closed (i.e., committed or rolled-back); they persist in memory until no longer referenced. Transaction.Callbacks may be registered but they will never be invoked.
