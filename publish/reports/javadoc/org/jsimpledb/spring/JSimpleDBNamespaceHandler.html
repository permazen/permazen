    public class JSimpleDBNamespaceHandler
extends NamespaceHandlerSupport
    Spring NamespaceHandler for the jsimpledb XML namespace.

    This class adds support for the <jsimpledb:jsimpledb>, <jsimpledb:scan-classes> and <jsimpledb:scan-field-types> XML tags.
