org.jsimpledb.annotation

Annotation Type JFieldType


  • @Retention(value=RUNTIME)
@Target(value=TYPE)
@Documented
public @interface JFieldType
    Java annotation for classes that define custom FieldTypes for a JSimpleDB's underlying Database. These classes will be automatically instantiated and registered with the associated FieldTypeRegistry.

    Annotated classes must extend FieldType and have a zero-parameter constructor.

    Note that once a certain encoding has been used for a given type name in a database, the encoding should not be changed without creating a new type (and type name), or changing the encoding signature. Otherwise, InconsistentDatabaseExceptions can result when the new type unexpectedly encounters the old encoding or vice-versa.
