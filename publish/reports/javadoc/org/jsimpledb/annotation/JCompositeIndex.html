org.jsimpledb.annotation

Annotation Type JCompositeIndex


  • @Retention(value=RUNTIME)
@Target(value={})
@Documented
public @interface JCompositeIndex
    Java annotation defining a composite index on a JSimpleClass-annotated class.

    A composite index is an index on two or more fields (to define a single-field index, just set JField.indexed() to true). All fields indexed in a composite index must be (a) simple and (b) not a sub-field of a complex field.

    See Also:
    JSimpleClass
