org.jsimpledb.annotation

Annotation Type Validate


  • @Retention(value=RUNTIME)
@Target(value=METHOD)
@Documented
public @interface Validate
    Annotates a JSimpleDB model class method that should be invoked any time the associated model object is validated.

    This annotation does not change when an object will be enqueued for validation. It only affects the behavior once validation of an instance is actually performed.

    When validating an object, validation via @Validate methods occurs after JSR 303 validation, if any.

    The annotated method must be an instance method (i.e., not static), return void, and take zero parameters. If validation fails, the method should throw a ValidationException.

    See Also:
    JTransaction.validate(), JObject.revalidate(), ValidationMode
