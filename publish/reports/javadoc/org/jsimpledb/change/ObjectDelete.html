  • Type Parameters:
    T - the type of the object that was deleted

    public class ObjectDelete<T>
extends Change<T>
    Change notification that indicates an object has been deleted.

    This type of change notification is never generated by JSimpleDB itself; object deletion notifications are instead delivered to @OnDelete methods, which do not take any parameters. This class exists as a convenience for application code that may want to unify handling of object change and object lifecycle events.
