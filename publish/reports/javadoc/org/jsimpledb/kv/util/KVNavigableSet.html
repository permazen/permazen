    public class KVNavigableSet
extends AbstractKVNavigableSet<byte[]>
    A NavigableSet view of the keys in a KVStore.

    Instances are mutable, with these exceptions:

    • Adding new elements via add() is not supported
    • clear() is not supported when a KeyFilter is configured
