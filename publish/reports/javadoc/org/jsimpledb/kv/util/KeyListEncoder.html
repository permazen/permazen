org.jsimpledb.kv.util

Class KeyListEncoder

  • java.lang.Object
    • org.jsimpledb.kv.util.KeyListEncoder

  • public final class KeyListEncoder
extends Object
    Serializes a sequence of byte[] arrays, compressing common prefixes.

    Keys are encoded in one of two forms:

    • total-length bytes...
    • -prefix-length suffix-length suffix-bytes ...
    The first length (total-length or negative prefix-length) is encoded using LongEncoder. The suffix-length, if present, is encoded using UnsignedIntEncoder.
