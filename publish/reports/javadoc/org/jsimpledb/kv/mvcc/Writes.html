org.jsimpledb.kv.mvcc

Class Writes

  • java.lang.Object
    • org.jsimpledb.kv.mvcc.Writes
  • All Implemented Interfaces:
    Mutations, SizeEstimating

    public class Writes
extends Object
implements Mutations, SizeEstimating
    Holds a set of writes to a KVStore.

    Each mutation is either a key/value put, the removal of a key range (possibly containing only a single key), or a counter adjustment.

    Instances are not thread safe.
