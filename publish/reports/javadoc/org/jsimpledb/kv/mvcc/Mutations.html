org.jsimpledb.kv.mvcc

Interface Mutations

  • All Known Implementing Classes:
    Writes

    public interface Mutations
    Represents a set of mutations that can be applied to a KVStore.

    Each mutation is either a key/value put, the removal of a key range (possibly containing only a single key), or a counter adjustment. Mutations are expected to be applied in the order: removes, puts, adjusts.
