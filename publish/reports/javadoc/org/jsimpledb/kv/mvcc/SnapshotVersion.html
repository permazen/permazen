Class SnapshotVersion

  • java.lang.Object
    • org.jsimpledb.kv.mvcc.SnapshotVersion

  • public class SnapshotVersion
extends Object
    Represents one SnapshotKVDatabase MVCC version.

    Information included:

    • MVCC version number
    • KVStore snapshot
    • Open transactions based on this version
    • The Writes of the transaction that was committed on this version, if any
