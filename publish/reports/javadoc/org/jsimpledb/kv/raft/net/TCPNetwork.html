Class TCPNetwork

    Network

    public class TCPNetwork
extends SelectorSupport
implements Network
    Network implementation based on TCP sockets.

    Remote peers have the String form IP-Address[:port]. If the port is omitted, DEFAULT_TCP_PORT is assumed.
