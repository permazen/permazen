053/**
054 * General purpose support superclass for Vaadin {@link com.vaadin.data.Container}s backed by {@link JSimpleDB} database objects.
055 * Automatically creates properties for object ID, database type, version, and fields, as well as any custom properties
056 * defined by {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated methods in Java model classes,
057 * and handles {@link Change} updates of backing objects or their related objects.
058 *
059 * <p>
060 * Instances are configured with a <b>type</b>, which can be any Java type (including interface types). The container
061 * will then be restricted to database objects that are instances of the configured type. The type may be null, in which
062 * case there is no restriction. The subclass method {@linkplain #queryForObjects queryForObjects()} determines which
063 * objects are actually loaded into the container. The items in the container are backed by in-memory copies of the
064 * corresponding database objects that live inside a {@link org.jsimpledb.SnapshotJTransaction}.
065 * </p>
066 *
067 * <p><b>Container Properties</b></p>
068 *
069 * <p>
070 * Instances will have the following container properties:
071 * <ul>
072 *  <li>{@link #OBJECT_ID_PROPERTY}: Object {@link ObjId}</li>
073 *  <li>{@link #TYPE_PROPERTY}: Object type name (JSimpleDB type name, not Java type name, though these are typically the same)</li>
074 *  <li>{@link #VERSION_PROPERTY}: Object schema version</li>
075 *  <li>{@link #REFERENCE_LABEL_PROPERTY}: Object <b>reference label</b>, which is a short description identifying the
076 *      object. Reference labels are used to provide "names" for objects that are more meaningful than object ID's
077 *      and are used as such in other {@link JSimpleDB} GUI classes. To customize the reference label for a Java model class,
078 *      annotate a method with {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}{@code (}{@link
079 *      JObjectContainer#REFERENCE_LABEL_PROPERTY REFERENCE_LABEL_PROPERTY}{@code )};
080 *      otherwise, the value of this property will be the same as {@link #OBJECT_ID_PROPERTY}.</li>
081 *  <li>A property for every {@link JSimpleDB} field that is common to all object types that sub-type
082 *      this instance's configured type. The property's ID is the field name; its value is as follows:
083 *      <ul>
084 *          <li>Simple fields will show their string values</li>
085 *          <li>Reference fields show the {@link #REFERENCE_LABEL_PROPERTY} of the referred-to object, or "Null"
086 *              if the reference is null</li>
087 *          <li>Set, list, and map fields show the first few entries in the collection</li>
088 *      </ul>
089 *  </li>
090 *  <li>A property for each {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated method
091 *      in the specified <b>type</b>. These properties will add to (or override) the properties listed above.
092 * </ul>
093 *
094 * <p><b>Loading</b></p>
095 *
096 * <p>
097 * Instances may be (re)loaded at any time by invoking {@link #reload}. This causes the container
098 * to query for {@link JObject}s within a new {@link JTransaction} via the subclass-provided method
099 * {@link #queryForObjects queryForObjects()}, which returns an {@link Iterable Iterable&lt;JObject&gt;}. This container
100 * copies the resulting database objects into the container's
101 * in-memory {@link org.jsimpledb.SnapshotJTransaction}. The latter effectively serves as the cache for the container,
102 * so that database transactions may be short-lived and are only required when (re)loading. Because the container contents
103 * are in memory, with large data sets this container should only be loaded with one "page" of objects at a time (a "page"
104 * typically being only what a human would be willing to scroll through at one time). It is up to
105 * {@link #queryForObjects queryForObjects()} to determine what and how many objects are returned, their sort order, etc.
106 * (if {@link #queryForObjects queryForObjects()} returns any objects that are not instances of the configured type,
107 * they are ignored). In any case, the maximum number of objects that will be loaded is limited by a
108 * {@linkplain #setMaxObjects configurable} maximum.
109 * </p>
110 *
111 * <p>
112 * When {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated methods are present on Java
113 * model classes, these property values may derive from other objects related to the backing instance. In order for these
114 * methods to work on the in-memory instance, the related objects must be copied into memory as well. The
115 * {@link #getDependencies getDependencies()} method allows the subclass to specify what other database objects
116 * some Vaadin property depends on.
117 * </p>
118 *
119 * <p><b>Updating Items</b></p>
120 *
121 * <p>
122 * Instances support updating individual items without requiring a complete container reload by invoking
123 * {@link #handleChange handleChange()}. In a typical design pattern, {@link org.jsimpledb.change.Change} objects originate
124 * from {@link org.jsimpledb.annotation.OnChange &#64;OnChange} methods and are broadcast (after copying into memory using
125 * a {@link org.jsimpledb.change.ChangeCopier}) to listeners such as this container by a transaction synchronization callback
126 * (see {@link org.jsimpledb.core.Transaction#addCallback Transaction.addCallback()}).
127 * </p>
128 *
129 * <p>
130 * Moreover, this container keeps track of which related objects are associated with each backing object (according to
131 * {@link #getDependencies getDependencies()}). {@link Change} events that affect a related object automatically cause
132 * the corresponding backing object(s) to update.
133 * </p>
134 *
135 * <p>
136 * This class responds to the creation or deletion of a database object matching this container's configured type
137 * by reloading the container; however, this is a conservative approach, as not necessarily every newly created (or
138 * deleted) object should be (or was) present in the container in the first place. Subclasses may wish to optimize this
139 * behavior by overriding {@link #handleObjectCreate handleObjectCreate()} and/or {@link #handleObjectDelete handleObjectDelete()}.
140 * </p>
141 *
142 * <p><b>{@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty} Limitations</b></p>
143 *
144 * <p>
145 * The use of {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty} methods has certain implications.
146 * First, if the method reads any of the object's field(s) via their Java getter methods (as would normally be expected),
147 * this will trigger a schema upgrade of the object if needed; however, this schema upgrade will occur in the
148 * container's in-memory {@link org.jsimpledb.SnapshotJTransaction} rather than in a real database transaction, so the
149 * {@link #VERSION_PROPERTY} will return a different schema version from what's in the database. The automatic schema
150 * upgrade can be avoided if desired by reading the field using the appropriate {@link JTransaction} field access method
151 * (e.g., {@link JTransaction#readSimpleField JTransaction.readSimpleField()}) and being prepared to handle a
152 * {@link org.jsimpledb.core.UnknownFieldException} if/when the object has an older schema version that does not contain
153 * the requested field.
154 * </p>
155 *
156 * <p>
157 * Secondly, because the values of reference fields (including complex sub-fields) are displayed using reference labels,
158 * and these are typically derived from the referenced object's fields, those indirectly referenced objects need to be
159 * copied into the container's {@link org.jsimpledb.SnapshotJTransaction} as well. The easiest way to ensure these indirectly
160 * referenced objects are copied is by overriding {@link #getDependencies getDependencies()} as described above. Alternately,
161 * the subclass may override {@link #copyOut copyOut()} to take complete control of copying objects into memory.
162 * </p>
163 */

295    /**
296     * Apply the given change to this container, if applicable.
297     *
298     * <p>
299     * The changed object {@linkplain Change#getObject referred to} by {@code change} must be accessible.
300     * Unless this method is invoked from within the transaction in which the change occurred, this means that
301     * {@code change} must have been be copied out of its transaction, e.g, using a {@link org.jsimpledb.change.ChangeCopier}.
302     * </p>
303     *
304     * <p>
305     * The implementation in {@link JObjectContainer} delegates to {@link #handleObjectCreate handleObjectCreate()},
306     * {@link #handleObjectDelete handleObjectDelete()}, or {@link #handleFieldChange handleFieldChange()} as appropriate.
307     * </p>
308     *
309     * @param change change from a transaction
310     * @param <T> type of the changed object
311     * @throws IllegalArgumentException if {@code change} is null
312     * @throws org.jsimpledb.core.StaleTransactionException if {@code change} refers to a {@link JObject}
313     *  without an associated tranasction.
314     */
315    public <T> void handleChange(Change<T> change) {
316
317        // Sanity check
318        if (change == null)
319            throw new IllegalArgumentException("null change");
320
321        // Determine whether the change concerns us
322        if (this.type != null && !this.type.isInstance(change.getObject()))
323            return;
324
325        // Apply change
326        change.visit(new ChangeAdapter<Void>() {
327
328            @Override
329            public <T> Void caseObjectDelete(ObjectDelete<T> change) {
330                JObjectContainer.this.handleObjectDelete(change);
331                return null;
332            }
333
334            @Override
335            public <T> Void caseObjectCreate(ObjectCreate<T> change) {
336                JObjectContainer.this.handleObjectCreate(change);
337                return null;
338            }
339
340            @Override
341            protected <T> Void caseFieldChange(FieldChange<T> change) {
342                JObjectContainer.this.handleFieldChange(change);
343                return null;
344            }
345        });
346    }

376    /**
377     * Handle a {@link FieldChange} event.
378     *
379     * <p>
380     * The implementation in {@link JObjectContainer} triggers value change events for all affected item(s) in this container.
381     * Affected items are those whose backing object is either the changed object, or has the changed object as a dependency
382     * according to {@link #getDependencies getDependencies()}.
383     * </p>
384     *
385     * @param change change from a transaction
386     * @param <T> type of the changed object
387     */
388    protected <T> void handleFieldChange(FieldChange<T> change) {
389
390        // Check dependencies for the changed object
391        final JObject target = change.getJObject();
392        final ObjId id = target.getObjId();
393        final ObjIdSet oldDependencies = this.dependenciesMap.get(id);
394        final ObjIdSet newDependencies = new ObjIdSet(Iterables.transform(
395          Iterables.filter(this.getDependencies(target), JObject.class), new Function<JObject, ObjId>() {
396            @Override
397            public ObjId apply(JObject jobj) {
398                return jobj.getObjId();
399            }
400        }));
401        final ObjIdSet addedDependencies = newDependencies.clone();
402        if (oldDependencies != null)
403            addedDependencies.removeAll(oldDependencies);
404
405        // If any dependencies have been added, we may not have previously included them in the container so reload to be safe
406        if (!addedDependencies.isEmpty()) {
407            this.reload();
408            return;
409        }
410
411        // Update the matching backing object, if any
412        final SimpleItem<JObject> item = (SimpleItem<JObject>)this.getItem(id);
413        if (item != null) {
414            target.copyTo(item.getObject().getTransaction(), null, new CopyState());
415            item.fireValueChange();
416        }
417
418        // Update any other affected objects that are dependent on the target object
419        final ObjIdSet affectedIds = dependenciesMap.get(id);
420        if (affectedIds != null) {
421            final ObjIdSet notifiedIds = new ObjIdSet();
422            notifiedIds.add(id);
423            for (ObjId affectedId : affectedIds) {
424                final SimpleItem<JObject> affectedItem = (SimpleItem<JObject>)this.getItem(affectedId);
425                if (affectedItem != null && !notifiedIds.contains(affectedId)) {
426                    affectedItem.fireValueChange();
427                    notifiedIds.add(affectedId);
428                }
429            }
430        }
431    }

433    // This method runs within a transactin
434    private void doReloadInTransaction() {
435
436        // Get objects from subclass
437        Iterable<? extends JObject> jobjs = this.queryForObjects();
438
439        // Filter out any instances of the wrong type
440        if (JObjectContainer.this.type != null) {
441            jobjs = Iterables.filter(jobjs, new Predicate<JObject>() {
442                @Override
443                public boolean apply(JObject jobj) {
444                    return JObjectContainer.this.type.isInstance(jobj);
445                }
446            });
447        }
448
449        // Filter out any duplicate objects using ObjId set
450        final CopyState copyState = new CopyState();
451
452        // Copy database objects out of the database transaction into my in-memory transaction as we load them
453        jobjs = Iterables.transform(jobjs, new Function<JObject, JObject>() {
454            @Override
455            public JObject apply(JObject jobj) {
456                return JObjectContainer.this.copyOut(jobj, copyState);
457            }
458        });
459
460        // Limit the total number of objects in the container
461        jobjs = Iterables.limit(jobjs, this.maxObjects);
462
463        // Now actually load them
464        this.dependenciesMap.clear();
465        this.load(jobjs);
466    }

468    /**
469     * Copy the given database object, and any related objects needed by any
470     * {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated methods,
471     * into the current {@link org.jsimpledb.SnapshotJTransaction}.
472     *
473     * <p>
474     * The implementation in {@link JObjectContainer} copies {@code jobj}, and all of {@code jobj}'s related objects returned
475     * by {@link #getDependencies getDependencies()}, via {@link JTransaction#copyTo(JTransaction, CopyState, Iterable)}.
476     * </p>
477     *
478     * @param target the object to copy, or null (ignored)
479     * @param copyState tracks what's already been copied
480     * @return the copy of {@code target} in the current {@link org.jsimpledb.SnapshotJTransaction},
481     *  or null if {@code target} is null
482     */
483    protected JObject copyOut(JObject target, CopyState copyState) {
484
485        // Ignore null
486        if (target == null)
487            return null;
488
489        // Copy out object
490        final ObjId targetId = target.getObjId();
491        final JTransaction jtx = JTransaction.getCurrent();
492        final SnapshotJTransaction sjtx = jtx.getSnapshotTransaction();
493        final JObject copy = target.copyTo(sjtx, null, copyState);
494
495        // Copy out (and track) related objects
496        Iterable<? extends JObject> dependencies = this.getDependencies(target);
497        if (dependencies != null) {
498            dependencies = Iterables.filter(dependencies, JObject.class);                       // filter out nulls
499            jtx.copyTo(sjtx, copyState, dependencies);
500            for (JObject jobj : dependencies) {
501                final ObjId id = jobj.getObjId();
502                ObjIdSet affectedIds = this.dependenciesMap.get(id);
503                if (affectedIds == null) {
504                    affectedIds = new ObjIdSet();
505                    this.dependenciesMap.put(id, affectedIds);
506                }
507                affectedIds.add(targetId);
508            }
509        }
510
511        // Done
512        return copy;
513    }

556    private Collection<PropertyDef<?>> buildPropertyDefs() {
557        final PropertyDefHolder pdefs = new PropertyDefHolder();
558
559        // Add properties shared by all JObjects
560        pdefs.setPropertyDef(this.refLabelPropertyDef);
561        pdefs.setPropertyDef(this.objIdPropertyDef);
562        pdefs.setPropertyDef(this.objTypePropertyDef);
563        pdefs.setPropertyDef(this.objVersionPropertyDef);
564
565        // Add properties for all fields common to all sub-types of our configured type
566        final SortedMap<Integer, JField> jfields = Util.getCommonJFields(this.jdb.getJClasses(this.type));
567        if (jfields != null) {
568            for (JField jfield : jfields.values())
569                pdefs.setPropertyDef(new ObjFieldPropertyDef(jfield.getStorageId(), jfield.getName()));
570        }
571
572        // Apply any @ProvidesProperty-annotated method properties, possibly overridding jfields
573        if (this.propertyScanner != null) {
574            for (PropertyDef<?> propertyDef : this.propertyScanner.getPropertyDefs())
575                pdefs.setPropertyDef(propertyDef);
576        }
577
578        // Done
579        return pdefs.values();
580    }

594    @SuppressWarnings("unchecked")
595    private static <V> V extractProperty(PropertyExtractor<?> propertyExtractor, PropertyDef<V> propertyDef, JObject jobj) {
596        try {
597            return ((PropertyExtractor<JObject>)propertyExtractor).getPropertyValue(jobj, propertyDef);
598        } catch (DeletedObjectException e) {
599            try {
600                return propertyDef.getType().cast(new SizedLabel("<i>Unavailable</i>", ContentMode.HTML));
601            } catch (ClassCastException e2) {
602                try {
603                    return propertyDef.getType().cast("(Unavailable)");
604                } catch (ClassCastException e3) {
605                    return null;
606                }
607            }
608        }
609    }

614     * Support superclass for {@link PropertyDef} implementations that derive the property value from a {@link JObject}.
615     */
616    public abstract static class ObjPropertyDef<T> extends PropertyDef<T> {
617
618        protected ObjPropertyDef(String name, Class<T> type) {
619            super(name, type);
620        }
621
622        public abstract T extract(JObject jobj);
623    }

679    /**
680     * Implements the {@link JObjectContainer#REFERENCE_LABEL_PROPERTY} property.
681     */
682    public static class RefLabelPropertyDef extends ObjPropertyDef<Component> {
683
684        public RefLabelPropertyDef() {
685            super(REFERENCE_LABEL_PROPERTY, Component.class);
686        }
687
688        @Override
689        public Component extract(JObject jobj) {
690            final ReferenceMethodInfoCache.PropertyInfo<?> propertyInfo
691              = ReferenceMethodInfoCache.getInstance().getReferenceMethodInfo(jobj.getClass());
692            if (propertyInfo == ReferenceMethodInfoCache.NOT_FOUND)
693                return new ObjIdPropertyDef().extract(jobj);
694            final Object value = JObjectContainer.extractProperty(
695              propertyInfo.getPropertyExtractor(), propertyInfo.getPropertyDef(), jobj);
696            if (value instanceof Component)
697                return (Component)value;
698            return new SizedLabel(String.valueOf(value));
699        }
700    }

702// ObjFieldPropertyDef
703
704    /**
705     * Implements a property reflecting the value of a {@link JSimpleDB} field.
706     */
707    public class ObjFieldPropertyDef extends ObjPropertyDef<Component> {
708
709        private static final int MAX_ITEMS = 3;
710
711        private final int storageId;
712
713        public ObjFieldPropertyDef(int storageId, String name) {
714            super(name, Component.class);
715            this.storageId = storageId;
716        }
717
718        @Override
719        public Component extract(final JObject jobj) {
720            final JField jfield = JObjectContainer.this.jdb.getJClass(jobj.getObjId()).getJField(this.storageId, JField.class);
721            try {
722                return jfield.visit(new JFieldSwitchAdapter<Component>() {
723
724                    @Override
725                    public Component caseJSimpleField(JSimpleField field) {
726                        return ObjFieldPropertyDef.this.handleValue(field.getValue(jobj));
727                    }
728
729                    @Override
730                    public Component caseJCounterField(JCounterField field) {
731                        return ObjFieldPropertyDef.this.handleValue(field.getValue(jobj).get());
732                    }
733
734                    @Override
735                    protected Component caseJCollectionField(JCollectionField field) {
736                        return ObjFieldPropertyDef.this.handleCollectionField(field.getValue(jobj));
737                    }
738
739                    @Override
740                    public Component caseJMapField(JMapField field) {
741                        return ObjFieldPropertyDef.this.handleMultiple(Iterables.transform(field.getValue(jobj).entrySet(),
742                          new Function<Map.Entry<?, ?>, Component>() {
743                            @Override
744                            public Component apply(Map.Entry<?, ?> entry) {
745                                final HorizontalLayout layout = new HorizontalLayout();
746                                layout.setMargin(false);
747                                layout.setSpacing(false);
748                                layout.addComponent(ObjFieldPropertyDef.this.handleValue(entry.getKey()));
749                                layout.addComponent(new SizedLabel(" \u21d2 "));        // RIGHTWARDS DOUBLE ARROW
750                                layout.addComponent(ObjFieldPropertyDef.this.handleValue(entry.getValue()));
751                                return layout;
752                            }
753                        }));
754                    }
755                });
756            } catch (UnknownFieldException e) {
757                return new SizedLabel("<i>NA</i>", ContentMode.HTML);
758            }
759        }
760
761        private Component handleCollectionField(Collection<?> col) {
762            return this.handleMultiple(Iterables.transform(col, new Function<Object, Component>() {
763                @Override
764                public Component apply(Object item) {
765                    return ObjFieldPropertyDef.this.handleValue(item);
766                }
767            }));
768        }
769
770        private Component handleMultiple(Iterable<Component> components) {
771            final HorizontalLayout layout = new HorizontalLayout();
772            layout.setMargin(false);
773            layout.setSpacing(false);
774            int count = 0;
775            for (Component component : components) {
776                if (count >= MAX_ITEMS) {
777                    layout.addComponent(new SizedLabel("..."));
778                    break;
779                }
780                if (count > 0)
781                    layout.addComponent(new SizedLabel(",&#160;", ContentMode.HTML));
782                layout.addComponent(component);
783                count++;
784            }
785            return layout;
786        }
787
788        @SuppressWarnings("unchecked")
789        private <T> Component handleValue(Object value) {
790            if (value == null)
791                return new SizedLabel("<i>Null</i>", ContentMode.HTML);
792            if (value instanceof JObject)
793                return new RefLabelPropertyDef().extract((JObject)value);
794            return new SizedLabel(String.valueOf(value));
795        }
796    }
