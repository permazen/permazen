026/**
027 * "Disruptor" style transaction manager for {@link PersistentObject} databases.
028 *
029 * <h3>Overview</h3>
030 *
031 * <p>
032 * Instances of this class maintain a dedicated thread for performing transactions on a specific {@link PersistentObject}
033 * database. The {@link #performTransaction performTransaction()} and {@link #scheduleTransaction scheduleTransaction()}
034 * methods schedule synchronous and asynchronous transactions (respectively) to be executed by this thread.
035 * </p>
036 *
037 * <p>
038 * By performing all transactions on the same thread, the {@link PersistentObject} root object is stays in the cache
039 * of a single CPU core, improving performance ("Disruptor" style). The single threading also guarantees transaction
040 * serialization, atomicity, and isolation. The capacity of the queue that holds upcoming transactions can be configured
041 * via {@link #setQueueCapacity setQueueCapacity()} (default {@link Integer#MAX_VALUE}). When the queue is full,
042 * scheduling a new transaction blocks until space is available.
043 * </p>
044 *
045 * <p>
046 * Within a transaction, {@link #getRoot} is used to access that transaction's {@link PersistentObject} root object.
047 * The returned object graph is private to that thread's transaction (except in shared mode; see below).
048 * </p>
049 *
050 * <h3>Transaction Types</h3>
051 *
052 * <p>
053 * Transactions may be read-only or read-write. Read-only transactions may optionally be configured to access a
054 * {@linkplain PersistentObject#getSharedRoot shared root}, in which case all read-only transactions share the same
055 * object graph (so no per-transation copies are required), but these transactions must also be careful not to modify
056 * the object graph returned by {@link #getRoot}. For read-write transactions, any modifications to the
057 * object graph returned by {@link #getRoot} will be persisted when the transaction completes (assuming it validates).
058 * In either case, references to the object graph returned by {@link #getRoot} may be returned to callers outside of the
059 * transaction, who may then safely access them with the assurance that they won't be modified by another thread (although in the
060 * {@linkplain PersistentObject#getSharedRoot shared root} case, modifications must still be avoided). Of course, once
061 * the transaction ends the transaction's object graph is no longer guaranteed to be up-to-date.
062 * </p>
063 *
064 * <p>
065 * Normally all changes to the associated {@link PersistentObject} database are being handled via this class: if not, and
066 * another thread updates the {@link PersistentObject} while a transaction is being handled,
067 * a {@link PersistentObjectVersionException} can result (the transaction may then be retried if desired).
068 * </p>
069 *
070 * <h3>Transaction Reentrancy</h3>
071 *
072 * <p>
073 * Transactions are reentrant, so that if {@link #performTransaction performTransaction()} is invoked by a thread while
074 * executing within an existing transaction, no new transaction is created; instead, execution proceeds in the context of
075 * the existing transaction. However, if the second transaction is not compatible with the first (e.g., a read-write
076 * transaction inside a read-only one), an exception is thrown.
077 * </p>
078 *
079 * <h3>Multiple Transactions</h3>
080 *
081 * <p>
082 * A thread executing within one {@link PersistentObjectTransactionManager}'s transaction may safely invoke
083 * {@link #performTransaction performTransaction()} on another {@link PersistentObjectTransactionManager}; however,
084 * the first transaction will block while the second executes, and callers must be careful to always nest these
085 * transactions in the same order to avoid possible deadlock.
086 * </p>
087 *
088 * <p>
089 * To allow distingushing among multiple {@link PersistentObjectTransactionManager} instances that may have open transactions
090 * at the same time, instances may be given unique {@linkplain #getName names} (names are automatically inherited from the
091 * {@linkplain #setBeanName bean name} when instances are declared in a Spring bean factory). Within an open transaction,
092 * the associated {@link PersistentObjectTransactionManager} can be accessed by name via {@link #getCurrent(String)}.
093 * As a convenience, in the common case of there only being one open transaction in the current thread, the
094 * {@link #getCurrent()} variant works; dealing with names is not necessary when there is only one instance in play.
095 * </p>
096 *
097 * <h3>Annotation Usage</h3>
098 *
099 * <p>
100 * The easiest way to use this class is to annotate transactional methods with the
101 * {@link PersistentObjectTransactional &#64;PersistentObjectTransactional} annotation and apply the associated AOP aspect,
102 * and then use {@link #getCurrent()} within the transaction to access the {@link PersistentObjectTransactionManager} and
103 * the associated transaction root object. For example:
104 * <pre>
105 *  <b>&#64;PersistentObjectTransactional</b>
106 *  public User loginUser(String username) throws LoginException {
107 *      final MyDatabase database = <b>PersistentObjectTransactionManager.&lt;MyDatabase&gt;getCurrent().getRoot()</b>;
108 *      for (User user : database.getUsers()) {
109 *          if (user.getUsername().equals(username)) {
110 *              if (user.getLoginCount() &gt;= MAX_CONCURRENT_LOGINS)
111 *                  throw new LoginException("too many logins");
112 *              user.setLoginCount(user.getLoginCount() + 1);
113 *              return user;
114 *          }
115 *      }
116 *      throw new LoginException("user not found");
117 *  }
118 * </pre>
119 * Don't forget to configure the {@link PersistentObjectTransactional &#64;PersistentObjectTransactional} AOP aspect
120 * via Spring as described in {@link PersistentObjectTransactional &#64;PersistentObjectTransactional}.
121 * </p>
122 *
123 * @param <T> root object type
124 * @see PersistentObjectTransactional &#64;PersistentObjectTransactional
125 */
126@Component
127public class PersistentObjectTransactionManager<T> extends AbstractBean implements BeanNameAware {

177    @Override
178    public void afterPropertiesSet() throws Exception {
179        super.afterPropertiesSet();
180
181        // Check required properties
182        if (this.persistentObject == null)
183            throw new IllegalStateException("no PersistentObject configured");
184        if (this.name == null)
185            throw new IllegalStateException("no name configured");
186
187        // Initialize queue
188        this.queue = new LinkedBlockingQueue<FutureTask<?>>(this.getQueueCapacity());
189
190        // Start worker thread
191        if (this.workerThread != null)
192            throw new IllegalStateException("instance is already started");
193        this.workerThread = new WorkerThread();
194        this.workerThread.start();
195    }
196
197    @Override
198    public void destroy() throws Exception {
199
200        // Stop worker thread
201        if (this.workerThread == null)
202            throw new IllegalStateException("instance is already shutdown");
203        this.queue.add(TERMINATE_TASK);
204
205        // Wait for worker thread to finish
206        this.workerThread.join();
207        this.workerThread = null;
208
209        // Reset queue
210        this.queue = null;
211
212        // Done
213        super.destroy();
214    }

282    /**
283     * Schedule a transaction, wait for it to complete, and return the result.
284     *
285     * <p>
286     * If the current thread is already executing within a transaction associated with this instance, then
287     * no new transaction is created; instead, the {@code action} executes in the context of the existing
288     * transaction.
289     * </p>
290     *
291     * @param action action to perform within the transaction
292     * @param readOnly true for a read-only transaction
293     * @param shared if {@code readOnly} is true, access the {@linkplain PersistentObject#getSharedRoot shared root}
294     *  to avoid a copy operation; in this case, {@code action} must not make any modifications
295     * @return result of executing the {@code action}
296     * @throws IllegalArgumentException if {@code action} is null
297     * @throws IllegalStateException if this instance is not yet {@linkplain #afterPropertiesSet started}
298     *  or has been {@linkplain #destroy} shut down
299     * @throws IllegalStateException if this instance already has a read-only transaction open in the current thread
300     *  and {@code readOnly} is false
301     * @throws IllegalStateException if this instance already has a shared mode read-only transaction open in the current thread
302     *  and {@code shared} is false
303     * @throws PersistentObjectValidationException if {@code readOnly} is false and the modified root does not validate
304     * @throws PersistentObjectVersionException if {@code readOnly} is false and the database was modified by another thread
305     * @throws PersistentObjectException if {@code readOnly} is false and some other error occurs
306     * @throws Exception if thrown by {@code action}
307     */
308    public <V> V performTransaction(Callable<V> action, boolean readOnly, boolean shared) throws Exception {
309        if (this.isReentrant(readOnly, shared))
310            return action.call();
311        return this.performTransaction(this.scheduleTransaction(action, readOnly, shared));
312    }

315     * Schedule a transaction and wait for it to complete.
316     *
317     * @param action action to perform within the transaction
318     * @param readOnly true for a read-only transaction
319     * @param shared if {@code readOnly} is true, access the {@linkplain PersistentObject#getSharedRoot shared root}
320     *  to avoid a copy operation; in this case, {@code action} must not make any modifications
321     * @throws IllegalArgumentException if {@code action} is null
322     * @throws IllegalStateException if this instance is not yet {@linkplain #afterPropertiesSet started}
323     *  or has been {@linkplain #destroy} shut down
324     * @throws IllegalStateException if this instance already has a read-only transaction open in the current thread
325     *  and {@code readOnly} is false
326     * @throws IllegalStateException if this instance already has a shared mode read-only transaction open in the current thread
327     *  and {@code shared} is false
328     * @throws PersistentObjectValidationException if {@code readOnly} is false and the modified root does not validate

332    public void performTransaction(Runnable action, boolean readOnly, boolean shared) {
333        if (this.isReentrant(readOnly, shared)) {
334            action.run();
335            return;
336        }
337        this.performTransaction(this.scheduleTransaction(action, readOnly, shared));
338    }

341     * Get the instance associated with an open transaction in the current thread and having the given manager name.
342     *
343     * @param name name assigned to the desired instance
344     * @throws IllegalArgumentException if {@code name} is null
345     * @throws IllegalStateException if the current thread is not running within a transaction managed by
346     *  a {@link PersistentObjectTransactionManager} assigned the given name
347     */
348    @SuppressWarnings("unchecked")
349    public static <T> PersistentObjectTransactionManager<T> getCurrent(String name) {
350        if (name == null)
351            throw new IllegalArgumentException("null name");
352        final HashMap<String, PersistentObjectTransactionManager<?>> managerMap = ASPECT_MANAGER_MAP.get();
353        final PersistentObjectTransactionManager<?> manager = managerMap != null ? managerMap.get(name) : null;
354        if (manager == null) {
355            throw new IllegalStateException("no PersistentObjectTransactionManager named `" + name
356              + "' has an open transaction in the current thread");
357        }
358        return (PersistentObjectTransactionManager<T>)manager;
359    }

361    /**
362     * Get the (unique) instance associated with an open transaction in the current thread.
363     *
364     * <p>
365     * This is a convenience method for the common case where there is only one instance associated with the current thread.
366     * </p>
367     *
368     * @throws IllegalStateException if the current thread is not running within a
369     *  {@link PersistentObjectTransactionManager} transaction
370     * @throws IllegalStateException there is more than one {@link PersistentObjectTransactionManager} transaction
371     *  open in the current thread
372     */
373    @SuppressWarnings("unchecked")
374    public static <T> PersistentObjectTransactionManager<T> getCurrent() {
375        final HashMap<String, PersistentObjectTransactionManager<?>> managerMap = ASPECT_MANAGER_MAP.get();
376        if (managerMap == null || managerMap.isEmpty()) {
377            throw new IllegalStateException("there are no PersistentObjectTransactionManager transactions open"
378              + " in the current thread");
379        }
380        if (managerMap.size() > 1) {
381            throw new IllegalStateException("there is more than one PersistentObjectTransactionManager transaction open"
382              + " in the current thread");
383        }
384        return (PersistentObjectTransactionManager<T>)managerMap.values().iterator().next();
385    }

431    private boolean isReentrant(boolean readOnly, boolean shared) {
432
433        // Are we already within a transaction?
434        final TxInfo txInfo = this.currentRoot.get();
435        if (txInfo == null)
436            return false;
437
438        // Check for incompatibilities
439        if (txInfo.isReadOnly()) {
440            if (!readOnly) {
441                throw new IllegalStateException("illegal read-write transaction created reentrantly"
442                  + " within a read-only transaction");
443            }
444            if (txInfo.isShared() && !shared) {
445                throw new IllegalStateException("illegal non-shared read-only transaction created reentrantly"
446                  + " within a shared read-only transaction");
447            }
448        }
449        return true;
450    }

452    private <V> FutureTask<V> scheduleTransaction(FutureTask<V> task) {
453
454        // Sanity check
455        if (task == null)
456            throw new IllegalArgumentException("null task");
457
458        // Enqueue transaction
459        try {
460            this.queue.add(task);
461        } catch (NullPointerException e) {
462            throw new IllegalStateException("instance not started or has been shutdown");
463        }
464
465        // Done
466        return task;
467    }
468
469    private <V> V performTransaction(Future<V> future) {
470        boolean interrupted = false;
471        try {
472            while (true) {
473                try {
474                    return future.get();
475                } catch (InterruptedException e) {
476                    interrupted = true;
477                }
478            }
479        } catch (ExecutionException e) {
480            final Throwable cause = e.getCause();
481            this.prependCurrentStackTrace(cause, "Synchronous PersistentObjectTransactionManager Transaction");
482            throw this.<RuntimeException>maskException(cause);                                      // re-throw original exception
483        } catch (CancellationException e) {
484            throw new RuntimeException("transaction method execution was canceled", e);             // should never happen
485        } finally {
486            if (interrupted)
487                Thread.currentThread().interrupt();
488        }
489    }
490
491    private <V> Callable<V> getTask(final Callable<V> action, final boolean readOnly, final boolean shared) {
492        return new Callable<V>() {
493            @Override
494            public V call() throws Exception {
495                final PersistentObject<T>.Snapshot snapshot = readOnly && shared ?
496                  PersistentObjectTransactionManager.this.persistentObject.getSharedRootSnapshot() :
497                  PersistentObjectTransactionManager.this.persistentObject.getRootSnapshot();
498                final TxInfo txInfo = new TxInfo(snapshot, readOnly, shared);
499                return PersistentObjectTransactionManager.this.currentRoot.invoke(txInfo, new Callable<V>() {
500                    @Override
501                    public V call() throws Exception {
502                        return PersistentObjectTransactionManager.this.executeTransaction(action, txInfo);
503                    }
504                });
505            }
506        };
507    }
508
509    private Callable<Void> getTask(final Runnable action, boolean readOnly, boolean shared) {
510        return this.getTask(new Callable<Void>() {
511            @Override
512            public Void call() {
513                action.run();
514                return null;
515            }
516        }, readOnly, shared);
517    }
518
519    private <V> V executeTransaction(Callable<V> action, TxInfo txInfo) throws Exception {
520
521        // Associate this instance with the current thread
522        HashMap<String, PersistentObjectTransactionManager<?>> managerMap = ASPECT_MANAGER_MAP.get();
523        final boolean emptyMap = managerMap == null;
524        if (emptyMap) {
525            managerMap = new HashMap<String, PersistentObjectTransactionManager<?>>();
526            ASPECT_MANAGER_MAP.set(managerMap);
527        }
528        if (managerMap.containsKey(this.name)) {
529            throw new IllegalStateException("A PersistentObjectTransactionManager named `" + this.name
530              + "' already has an open transaction in the current thread; all names must be distinct");
531        }
532        managerMap.put(this.name, this);
533
534        // Perform transaction
535        V result;
536        try {
537
538            // Invoke callback
539            try {
540                result = action.call();
541            } catch (Exception e) {
542                throw e;
543            }
544
545            // Update database (read-write transactions only)
546            if (!txInfo.isReadOnly())
547                this.persistentObject.setRoot(txInfo.getSnapshot().getRoot(), txInfo.getSnapshot().getVersion());
548        } finally {
549            if (emptyMap)
550                ASPECT_MANAGER_MAP.remove();
551            else
552                managerMap.remove(this.name);
553        }
554
555        // Done
556        return result;
557    }

570    // Prepend the second exception's stack frames onto the first exception's stack frames
571    void prependStackFrames(Throwable t, Throwable p, String description, int skip) {
572        final StackTraceElement[] innerFrames = t.getStackTrace();
573        final StackTraceElement[] outerFrames = p.getStackTrace();
574        final StackTraceElement[] frames = new StackTraceElement[innerFrames.length + 1 + Math.max(outerFrames.length - skip, 0)];
575        System.arraycopy(innerFrames, 0, frames, 0, innerFrames.length);
576        frames[innerFrames.length] = new StackTraceElement(this.getClass().getName(), "<" + this.name + ">", description, -1);
577        for (int i = 0; i < outerFrames.length - skip; i++)
578            frames[innerFrames.length + 1 + i] = outerFrames[i + skip];
579        t.setStackTrace(frames);
580    }

582// Worker thread
583
584    private class WorkerThread extends Thread {
585
586        private final Logger log = LoggerFactory.getLogger(this.getClass());
587
588        public WorkerThread() {
589            super("PersistentObjectTransactionManager-" + PersistentObjectTransactionManager.THREAD_COUNTER.incrementAndGet());
590        }
591
592        @Override
593        public void run() {
594            this.log.debug(this + " transaction thread starting");
595            int count = 0;
596            try {
597                while (true) {
598
599                    // Get next in queue
600                    final FutureTask<?> task = PersistentObjectTransactionManager.this.queue.take();
601                    if (task == TERMINATE_TASK)
602                        break;
603
604                    // Execute task
605                    task.run();
606                    count++;
607                }
608            } catch (Throwable t) {
609                this.log.error(this + " transaction thread caught unexpected exception", t);
610            } finally {
611                this.log.debug(this + " transaction thread stopping after " + count + " transactions");
612            }
613        }
614    }
