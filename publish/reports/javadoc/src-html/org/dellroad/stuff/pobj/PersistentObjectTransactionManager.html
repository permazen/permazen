025/**
026 * "Disruptor" style transaction manager for {@link PersistentObject} databases.
027 *
028 * <h3>Overview</h3>
029 *
030 * <p>
031 * Instances of this class manage transactions on a {@link PersistentObject} database and support the
032 * {@link PersistentObjectTransactional &#64;PersistentObjectTransactional} annotation and related AOP aspect.
033 * All transactions are serialized, atomic, and isolated. Because they are serialized, failures due to
034 * optimistic locking are not possible. Instances operate in one of two modes: in normal mode, transactions
035 * are executed on the current thread; in "Disruptor" mode, transactions are executed on a dedicated transaction thread.
036 * In either case, {@link #performTransaction performTransaction()} performs a synchronous transaction. In Disruptor mode
037 * {@link #scheduleTransaction scheduleTransaction()} can be also used, to schedule an asynchronous transaction.
038 * </p>
039 *
040 * <p>
041 * In Disruptor mode, by performing all transactions on the same thread, all writes to the {@link PersistentObject} root
042 * object stay in the cache of a single CPU core, improving performance.  The capacity of the queue that holds upcoming
043 * transactions can be configured via {@link #setQueueCapacity setQueueCapacity()} (default {@link Integer#MAX_VALUE}).
044 * When the queue is full, scheduling a new transaction blocks until space is available.
045 * </p>
046 *
047 * <p>
048 * Within a transaction, {@link #getRoot} is used to access that transaction's {@link PersistentObject} root object.
049 * The returned object graph is private to that thread's transaction (except in read-only shared mode; see below).
050 * </p>
051 *
052 * <h3>Transaction Types</h3>
053 *
054 * <p>
055 * Transactions may be read-only or read-write. Read-only transactions may optionally be configured to access a
056 * {@linkplain PersistentObject#getSharedRoot shared root}, in which case all read-only transactions share the same
057 * object graph (so no per-transation copies are required), but these transactions must also be careful not to modify
058 * the object graph returned by {@link #getRoot}. In non-shared read-only transactions, changes are permitted but
059 * discarded at the end of the transaction. For read-write transactions, any modifications to the
060 * object graph returned by {@link #getRoot} will be persisted when the transaction completes (assuming it validates).
061 * In either case, references to the object graph returned by {@link #getRoot} may be returned to callers outside of the
062 * transaction, who may then safely access them with the assurance that they won't be modified by another thread (although in the
063 * {@linkplain PersistentObject#getSharedRoot shared root} case, modifications must still be avoided). Of course, once
064 * the transaction ends, the transaction's object graph is also no longer guaranteed to be up-to-date.
065 * </p>
066 *
067 * <p>
068 * Normally all changes to the associated {@link PersistentObject} database are being handled via this class: if not,
069 * and another thread updates the {@link PersistentObject} while a transaction is being handled,
070 * a {@link PersistentObjectVersionException} can result (the transaction may then be retried if desired).
071 * </p>
072 *
073 * <h3>Transaction Reentrancy</h3>
074 *
075 * <p>
076 * Transactions are reentrant, so that if {@link #performTransaction performTransaction()} is invoked by a thread while
077 * executing within an existing transaction, no new transaction is created; instead, execution proceeds in the context of
078 * the existing transaction. However, if the second transaction is not compatible with the first (e.g., a read-write
079 * transaction inside a read-only one), an exception is thrown.
080 * </p>
081 *
082 * <h3>Multiple Transactions</h3>
083 *
084 * <p>
085 * A thread executing within one {@link PersistentObjectTransactionManager}'s transaction may safely invoke
086 * {@link #performTransaction performTransaction()} on another {@link PersistentObjectTransactionManager}; however,
087 * the first transaction will block while the second executes, and therefore callers must be careful to always nest
088 * these transactions in a consistent ordering to avoid possible deadlock.
089 * </p>
090 *
091 * <p>
092 * To allow distingushing among multiple {@link PersistentObjectTransactionManager} instances that may have open transactions
093 * at the same time, instances may be given unique {@linkplain #getName names} (names are automatically inherited from the
094 * {@linkplain #setBeanName bean name} when instances are declared in a Spring bean factory). Within an open transaction,
095 * the associated {@link PersistentObjectTransactionManager} can be accessed by name via {@link #getCurrent(String)}.
096 * As a convenience, in the common case of there only being one open transaction in the current thread, the
097 * {@link #getCurrent()} variant works; dealing with names is not necessary when there is only one instance.
098 * </p>
099 *
100 * <h3>Annotation Usage</h3>
101 *
102 * <p>
103 * The easiest way to use this class is to annotate transactional methods with the
104 * {@link PersistentObjectTransactional &#64;PersistentObjectTransactional} annotation and apply the associated AOP aspect,
105 * and then use {@link #getCurrent()} within the transaction to access the {@link PersistentObjectTransactionManager} and
106 * the associated transaction root object. For example:
107 * <pre>
108 *  // Login a user. This method operates on the database atomically.
109 *  <b>&#64;PersistentObjectTransactional</b>
110 *  public User loginUser(String username) throws LoginException {
111 *      final MyDatabase database = <b>PersistentObjectTransactionManager.&lt;MyDatabase&gt;getCurrent().getRoot()</b>;
112 *      for (User user : database.getUsers()) {
113 *          if (user.getUsername().equals(username)) {
114 *              if (user.getLoginCount() &gt;= MAX_CONCURRENT_LOGINS)
115 *                  throw new LoginException("too many logins");
116 *              user.setLoginCount(user.getLoginCount() + 1);
117 *              return user;
118 *          }
119 *      }
120 *      throw new LoginException("user not found");
121 *  }
122 *
123 *  // Grab a read-only database snapshot.
124 *  <b>&#64;PersistentObjectTransactional(readOnly = true, shared = true)</b>
125 *  public MyDatabase getMyDatabase() {
126 *      return <b>PersistentObjectTransactionManager.&lt;MyDatabase&gt;getCurrent().getRoot()</b>;
127 *  }
128 * </pre>
129 * Don't forget to configure the {@link PersistentObjectTransactional &#64;PersistentObjectTransactional} AOP aspect
130 * via Spring as described in {@link PersistentObjectTransactional &#64;PersistentObjectTransactional}.
131 * </p>
132 *
133 * @param <T> root object type
134 * @see PersistentObjectTransactional &#64;PersistentObjectTransactional
135 * @see <a href="http://lmax-exchange.github.io/disruptor/">LMAX Disruptor: High Performance Inter-Thread Messaging Library</a>
136 */

180    /**
181     * Start this instance.
182     *
183     * @throws IllegalStateException if instance has already been started.
184     */
185    @Override
186    public void afterPropertiesSet() throws Exception {
187        super.afterPropertiesSet();
188        synchronized (this) {
189
190            // Sanity check
191            if (this.started)
192                throw new IllegalStateException("already started");
193
194            // Check required properties
195            if (this.persistentObject == null)
196                throw new IllegalStateException("no PersistentObject configured");
197            if (this.name == null)
198                throw new IllegalStateException("no name configured");
199
200            // Check mode
201            if (this.disruptorMode) {
202
203                // Initialize queue
204                this.queue = new LinkedBlockingQueue<FutureTask<?>>(this.getQueueCapacity());
205
206                // Start worker thread
207                this.workerThread = new WorkerThread();
208                this.workerThread.start();
209            }
210
211            // Done
212            this.started = true;
213        }
214    }

216    /**
217     * Stop this instance.
218     *
219     * @throws IllegalStateException if instance has not yet been {@linkplain #afterPropertiesSet started}.
220     */
221    @Override
222    public void destroy() throws Exception {
223        synchronized (this) {
224
225            // Sanity check
226            if (!this.started)
227                throw new IllegalStateException("not started");
228
229            // Stop worker thread
230            if (this.queue != null)
231                this.queue.add(TERMINATE_TASK);
232
233            // Wait for worker thread to finish
234            if (this.workerThread != null) {
235                this.workerThread.join();
236                this.workerThread = null;
237            }
238
239            // Reset queue
240            this.queue = null;
241
242            // Done
243            this.started = false;
244        }
245        super.destroy();
246    }

344    /**
345     * Perform a transaction synchronously.
346     *
347     * <p>
348     * In normal mode, the transaction executes in the current thread.
349     * In Disruptor mode, schedule a transaction, wait for it to complete, and return the result.
350     * <p>
351     *
352     * <p>
353     * If the current thread is already executing within a transaction associated with this instance, then
354     * no new transaction is created; instead, the {@code action} executes in the context of the existing
355     * transaction.
356     * </p>
357     *
358     * @param action action to perform within the transaction
359     * @param readOnly true for a read-only transaction
360     * @param shared if {@code readOnly} is true, access the {@linkplain PersistentObject#getSharedRoot shared root}
361     *  to avoid a copy operation; in this case, {@code action} must not make any modifications
362     * @return result of executing the {@code action}
363     * @throws IllegalArgumentException if {@code action} is null
364     * @throws IllegalStateException if this instance is not yet {@linkplain #afterPropertiesSet started}
365     *  or has been {@linkplain #destroy} shut down
366     * @throws IllegalStateException if this instance already has a read-only transaction open in the current thread
367     *  and {@code readOnly} is false
368     * @throws IllegalStateException if this instance already has a shared mode read-only transaction open in the current thread
369     *  and {@code shared} is false
370     * @throws PersistentObjectValidationException if {@code readOnly} is false and the modified root does not validate
371     * @throws PersistentObjectVersionException if {@code readOnly} is false and the database was modified by another thread
372     * @throws PersistentObjectException if {@code readOnly} is false and some other error occurs
373     * @throws Exception if thrown by {@code action}
374     */
375    public <V> V performTransaction(Callable<V> action, boolean readOnly, boolean shared) throws Exception {
376
377        // Check if reentrant
378        if (this.checkReentrant(readOnly, shared))
379            return action.call();
380
381        // Handle non-disruptor mode
382        if (!this.disruptorMode) {
383            final Callable<V> task = this.getTask(action, readOnly, shared);
384            synchronized (this.nonDisruptorLock) {
385                return task.call();
386            }
387        }
388
389        // Handle disruptor mode
390        return this.waitFormScheduledTransaction(this.scheduleTransaction(action, readOnly, shared));
391    }

393    /**
394     * Perform a transaction synchronously.
395     *
396     * <p>
397     * In normal mode, the transaction executes in the current thread.
398     * In Disruptor mode, schedule a transaction, wait for it to complete, and return the result.
399     * <p>
400     *
401     * @param action action to perform within the transaction
402     * @param readOnly true for a read-only transaction
403     * @param shared if {@code readOnly} is true, access the {@linkplain PersistentObject#getSharedRoot shared root}
404     *  to avoid a copy operation; in this case, {@code action} must not make any modifications
405     * @throws IllegalArgumentException if {@code action} is null
406     * @throws IllegalStateException if this instance is not yet {@linkplain #afterPropertiesSet started}
407     *  or has been {@linkplain #destroy} shut down
408     * @throws IllegalStateException if this instance already has a read-only transaction open in the current thread
409     *  and {@code readOnly} is false
410     * @throws IllegalStateException if this instance already has a shared mode read-only transaction open in the current thread
411     *  and {@code shared} is false
412     * @throws PersistentObjectValidationException if {@code readOnly} is false and the modified root does not validate
413     * @throws PersistentObjectVersionException if {@code readOnly} is false and the database was modified by another thread
414     * @throws PersistentObjectException if {@code readOnly} is false and some other error occurs
415     */
416    public void performTransaction(Runnable action, boolean readOnly, boolean shared) {
417
418        // Check if reentrant
419        if (this.checkReentrant(readOnly, shared)) {
420            action.run();
421            return;
422        }
423
424        // Handle non-disruptor mode
425        if (!this.disruptorMode) {
426            final Callable<Void> task = this.getTask(action, readOnly, shared);
427            try {
428                synchronized (this.nonDisruptorLock) {
429                    task.call();
430                }
431            } catch (Error e) {
432                throw e;
433            } catch (RuntimeException e) {
434                throw e;
435            } catch (Exception e) {
436                throw this.<RuntimeException>maskException(e);      // should never happen
437            }
438        }
439
440        // Handle disruptor mode
441        this.waitFormScheduledTransaction(this.scheduleTransaction(action, readOnly, shared));
442    }

457        final PersistentObjectTransactionManager<?> manager = managerMap != null ? managerMap.get(name) : null;
458        if (manager == null) {
459            throw new IllegalStateException("no PersistentObjectTransactionManager named `" + name
460              + "' has an open transaction in the current thread");
461        }
462        return (PersistentObjectTransactionManager<T>)manager;
463    }

535    private boolean checkReentrant(boolean readOnly, boolean shared) {
536
537        // Are we already within a transaction?
538        final TxInfo txInfo = this.currentRoot.get();
539        if (txInfo == null)
540            return false;
541
542        // Check for incompatibilities
543        if (txInfo.isReadOnly()) {
544            if (!readOnly) {
545                throw new IllegalStateException("illegal read-write transaction created reentrantly"
546                  + " within a read-only transaction");
547            }
548            if (txInfo.isShared() && !shared) {
549                throw new IllegalStateException("illegal non-shared read-only transaction created reentrantly"
550                  + " within a shared read-only transaction");
551            }
552        }
553        return true;
554    }

556    private <V> FutureTask<V> scheduleTransactionTask(FutureTask<V> task) {
557
558        // Sanity check
559        if (task == null)
560            throw new IllegalArgumentException("null task");
561        if (!this.started)
562            throw new IllegalStateException("instance not started");
563        if (this.queue == null)
564            throw new IllegalStateException("instance is not running in Disruptor mode");
565
566        // Enqueue transaction
567        this.queue.add(task);
568
569        // Done
570        return task;
571    }

573    private <V> V waitFormScheduledTransaction(Future<V> future) {
574        boolean interrupted = false;
575        try {
576            while (true) {
577                try {
578                    return future.get();
579                } catch (InterruptedException e) {
580                    interrupted = true;
581                }
582            }
583        } catch (ExecutionException e) {
584            final Throwable cause = e.getCause();
585            this.prependCurrentStackTrace(cause, "Synchronous PersistentObjectTransactionManager Transaction");
586            throw this.<RuntimeException>maskException(cause);                                      // re-throw original exception
587        } catch (CancellationException e) {
588            throw new RuntimeException("transaction method execution was canceled", e);             // should never happen
589        } finally {
590            if (interrupted)
591                Thread.currentThread().interrupt();
592        }
593    }

595    private <V> Callable<V> getTask(final Callable<V> action, final boolean readOnly, final boolean shared) {
596        return new Callable<V>() {
597            @Override
598            public V call() throws Exception {
599                final PersistentObject<T>.Snapshot snapshot = readOnly && shared ?
600                  PersistentObjectTransactionManager.this.persistentObject.getSharedRootSnapshot() :
601                  PersistentObjectTransactionManager.this.persistentObject.getRootSnapshot();
602                final TxInfo txInfo = new TxInfo(snapshot, readOnly, shared);
603                return PersistentObjectTransactionManager.this.currentRoot.invoke(txInfo, new Callable<V>() {
604                    @Override
605                    public V call() throws Exception {
606                        return PersistentObjectTransactionManager.this.executeTransaction(action, txInfo);
607                    }
608                });
609            }
610        };
611    }

613    private Callable<Void> getTask(final Runnable action, boolean readOnly, boolean shared) {
614        return this.getTask(new Callable<Void>() {
615            @Override
616            public Void call() {
617                action.run();
618                return null;
619            }
620        }, readOnly, shared);
621    }

623    private <V> V executeTransaction(Callable<V> action, TxInfo txInfo) throws Exception {
624
625        // Associate this instance with the current thread
626        HashMap<String, PersistentObjectTransactionManager<?>> managerMap = ASPECT_MANAGER_MAP.get();
627        final boolean emptyMap = managerMap == null;
628        if (emptyMap) {
629            managerMap = new HashMap<String, PersistentObjectTransactionManager<?>>();
630            ASPECT_MANAGER_MAP.set(managerMap);
631        }
632        if (managerMap.containsKey(this.name)) {
633            throw new IllegalStateException("A PersistentObjectTransactionManager named `" + this.name
634              + "' already has an open transaction in the current thread; all names must be distinct");
635        }
636        managerMap.put(this.name, this);
637
638        // Perform transaction
639        V result;
640        try {
641
642            // Invoke callback
643            try {
644                result = action.call();
645            } catch (Exception e) {
646                throw e;
647            }
648
649            // Update database (read-write transactions only)
650            if (!txInfo.isReadOnly())
651                this.persistentObject.setRoot(txInfo.getSnapshot().getRoot(), txInfo.getSnapshot().getVersion());
652        } finally {
653            if (emptyMap)
654                ASPECT_MANAGER_MAP.remove();
655            else
656                managerMap.remove(this.name);
657        }
658
659        // Done
660        return result;
661    }

664    @SuppressWarnings("unchecked")
665    <T extends Throwable> T maskException(Throwable t) throws T {
666        throw (T)t;

669    // Prepend stack frames from the current thread onto the given exception's stack frames
670    void prependCurrentStackTrace(Throwable t, String description) {
671        this.prependStackFrames(t, new Throwable(), description, 1);

674    // Prepend the second exception's stack frames onto the first exception's stack frames
675    void prependStackFrames(Throwable t, Throwable p, String description, int skip) {
676        final StackTraceElement[] innerFrames = t.getStackTrace();
677        final StackTraceElement[] outerFrames = p.getStackTrace();
678        final StackTraceElement[] frames = new StackTraceElement[innerFrames.length + 1 + Math.max(outerFrames.length - skip, 0)];
679        System.arraycopy(innerFrames, 0, frames, 0, innerFrames.length);
680        frames[innerFrames.length] = new StackTraceElement(this.getClass().getName(), "<" + this.name + ">", description, -1);
681        for (int i = 0; i < outerFrames.length - skip; i++)
682            frames[innerFrames.length + 1 + i] = outerFrames[i + skip];
683        t.setStackTrace(frames);
684    }

686// Worker thread
687
688    private class WorkerThread extends Thread {
689
690        private final Logger log = LoggerFactory.getLogger(this.getClass());
691
692        public WorkerThread() {
693            super("PersistentObjectTransactionManager-" + PersistentObjectTransactionManager.THREAD_COUNTER.incrementAndGet());
694        }
695
696        @Override
697        public void run() {
698            this.log.debug(this + " transaction thread starting");
699            int count = 0;
700            try {
701                while (true) {
702
703                    // Get next in queue
704                    final FutureTask<?> task = PersistentObjectTransactionManager.this.queue.take();
705                    if (task == TERMINATE_TASK)
706                        break;
707
708                    // Execute task
709                    task.run();
710                    count++;
711                }
712            } catch (Throwable t) {
713                this.log.error(this + " transaction thread caught unexpected exception", t);
714            } finally {
715                this.log.debug(this + " transaction thread stopping after " + count + " transactions");
716            }
717        }
718    }
