Project Modules

This project has declared the following modules:

Name Description
Permazen Ant Tasks Ant tasks for use with Permazen.
Permazen CLI Classes Permazen classes supporting command line interfaces.
Permazen CLI Application Permazen Command Line Interface (CLI) executable application.
Permazen Core API Classes Permazen core API classes which provide objects, fields, indexes, queries, and schema management on top of a key/value store.
Permazen Encoding Classes Permazen classes for encoding Java values to/from binary representations.
Permazen Key/Value Database Consistency Checker Permazen analog to UNIX fsck(8) command for checking the consistency of, and repairing any corruption in, a Permazen key/value database
Permazen Array Key/Value Store Permazen "array" key/value store implementations.
Permazen Berkeley DB Key/Value Store Permazen key/value store implementation based on Berkeley DB Java Edition.
Permazen Key/Value Caching Layer Permazen key/value store classes related to caching.
Permazen CockroachDB Key/Value Store Permazen key/value store implementation based on CockroachDB.
Permazen FoundationDB Key/Value Store Permazen key/value store implementation based on FoundationDB.
Permazen LevelDB Key/Value Store Permazen key/value store implementation based on LevelDB.
Permazen LMDB Key/Value Store Permazen key/value store implementation based on LMDB.
Permazen Microsoft SQL Server Key/Value Store Permazen key/value store based on Microsoft SQL Server.
Permazen H2 MVStore Key/Value Store Permazen key/value store implementation based on H2 Database MVStore.
Permazen MySQL Key/Value Store Permazen key/value store based on MySQL.
Permazen Raft Key/Value Store Permazen distributed, linearizable ACID compliant key/value store implementation based on the Raft consensus algorithm.
Permazen Simple Key/Value Store Classes A couple of simplistic Permazen key/value store implementations.
Permazen Google Cloud Spanner Key/Value Store Permazen key/value store implementation based on Google Cloud Spanner.
Permazen SQL Database Key/Value Store Support for Permazen key/value store implementation based on SQL databases.
Permazen SQLite Key/Value Store Permazen key/value store implementation based on SQLite.
Permazen Key/Value Store Test Classes Permazen unit test classes for key/value store implementations.
Permazen Xodus Key/Value Store Permazen key/value store implementation based on Xodus.
Permazen Key/Value Store Base Classes Permazen classes relating to the key/value store API.
Permazen Main Classes Permazen classes that map Java model classes onto the core API.
Permazen Maven Plugin Maven plugin for use with Permazen.
Permazen Spring Classes Permazen classes for the Spring Framework.
Permazen Test Classes Permazen unit test support classes.
Permazen Utility Classes Common utility classes used by Permazen.
